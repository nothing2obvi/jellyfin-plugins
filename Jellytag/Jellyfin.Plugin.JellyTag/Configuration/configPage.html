Performance
Use 0 to keep cached images forever.
When cache duration has expired, JellyTag-Plus can check whether the same image and badges would still be used. If nothing changed, it keeps the cached image and resets its age instead of rendering it again. Shortcut cache hits stay fast and skip this metadata work.
Estimated Warmer Progress: Based on variants completed by the cache warmer. Images rendered only by normal client browsing may be counted after the warmer touches them. Refresh this page to update progress.
Variable-size clients are covered by Learned Clients after real browsing records their Primary and Thumb requests. Jellyfin Web, Android, iOS, Desktop Qt, Tizen, WebOS, and Xbox calculate home and library sizes through Jellyfin Web. Wholphin, Moonfin Mobile-Desktop, Moonfin tvOS, Moonfin Smart-TV, DUNE, Swiftfin, and Jellyfin Desktop each have some fixed variants listed here, but their dynamic, grid, layout, or screen-derived variants are also handled by Learned Clients. Learned non-episode and non-video variants are grouped as Home & Libraries when the request does not reliably separate home from library browsing.
Advanced: when visible badges change for an image request, temporarily touches the real Jellyfin image and restores it so clients are more likely to request refreshed artwork.
